
# Magic 8 Ball +

**Projekta apraksts**
Magic 8 Ball + ir Python programma, kas palīdz lietotājam pieņemt lēmumus. Lietotājs var ievadīt vairākas izvēles un piešķirt katrai izvēlei svaru. Programma nejauši izvēlas vienu opciju, ņemot vērā norādītos svarus (lielāks svars = lielāka iespēja).
Šī programma ir uzlabota klasiskās Magic 8 Ball versijas ideja, jo tā ļauj personalizēt izvēles un saglabāt tās vēlākai izmantošanai.
** Programmas mērķis**
Programmas mērķis ir izveidot vienkāršu un intuitīvu rīku lēmumu pieņemšanai, izmantojot grafisko lietotāja saskarni (GUI) un nejaušas izvēles algoritmu ar svaru sistēmu.
**Funkcija**
izvēļu pievienošana
svaru piešķiršana katrai izvēlei
nejauša izvēle, ņemot vērā svarus
izvēļu saglabāšana failā
izvēļu ielāde no faila
grafiskā lietotāja saskarne GUI
