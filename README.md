# Rutarr
Magic 8 Ball +

Šī ir Python programma, kas palīdz pieņemt lēmumus.
Lietotājs var ievadīt izvēles un piešķirt tām svaru.
Programma nejauši izvēlas vienu opciju, ņemot vērā svarus.
Funkcijas:
izvēļu pievienošana
svaru sistēma
nejauša izvēle
datu saglabāšana
datu ielāde
(Izveidots Python valodā, izmantojot Tkinter bibliotēku).
